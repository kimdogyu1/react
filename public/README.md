# 포트폴리오

퍼블리셔업무 프로젝트 및 작업물 정리

- 그룹웨어
  - 채용 사이트
  - 서약서 및 예약 시스템
  - EMS서비스
  - connect 내부망 페이지 유지보수
- 기업홈페이지
  - 네이버 자회사 기업홈페이지 유지보수
- 쇼핑&프로모션
  - 쇼핑페스타, 네이버페이, 해피빈, 뮤직어워드 등

## 작업구성

사용된 gulp.js

- sass
- spritesmith
- watch
- sourcemaps
- tinify

설치
```ruby
npm install
```

실행
```ruby
gulp
```

컴포넌트 분리

```ruby
header.scss
footer.scss
variables.scss
mixin.scss
index.scss
```
